# Library JS Project

A JavaScript library app built as part of The Odin Project.

This project was created to practice JavaScript objects, constructor functions, prototypes, DOM manipulation, event handling, and rendering data from an array.

## Live Demo

https://miguelrcol.github.io/Library-js-project/

## Features

- Add new books with title, author, pages, and read status.
- Display books as cards on the page.
- Toggle each book between read and unread.
- Delete individual books.
- Use a unique ID for each book.
- Use a constructor function and prototype method.

## What I Learned

In this project, I practiced how to create objects using a constructor function, store them in an array, and render them dynamically in the DOM.

I also learned that the array is the source of truth, while the HTML cards are only the visual representation of that data.

## Difficulties

The hardest part was understanding how prototypes work and how to organize the logic between the data, the DOM, and the event listeners.

I also used AI assistance, but I want to become less dependent on it by rewriting and explaining more of the code myself.
